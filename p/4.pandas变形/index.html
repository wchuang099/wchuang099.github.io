长宽表的变形

pivot将长表转为宽表（单个聚合函数）

功能：将一列作为新列名，一列作为新行索引，填充值来自另一列。
限制：索引和列的组合必须唯一（不能有重复）。

语法：df.pivot(index='行索引', columns='列名', values='值')

import pandas as pd

# 原始长表
df_long = pd.DataFrame({
    '学生': ['小明', '小明', '小红', '小红'],
    '科目': ['数学', '语文', '数学', '语文'],
    '成绩': [85, 90, 78, 92]
})

print("原始长表：")
print(df_long)

# 转为宽表
df_wide = df_long.pivot(index='学生', columns='科目', values='成绩')
print("\npivot 转宽表：")
print(df_wide)

# 宽表
df_wide = pd.DataFrame({
    '学生': ['小明', '小红'],
    '数学': [85, 78],
    '语文': [90, 92]
})

# 转为长表
df_melt = df_wide.melt(
    id_vars='学生',
    value_vars=['数学', '语文'],
    var_name='科目',
    value_name='成绩'
)

df = pd.DataFrame(
    [[85, 90], [78, 92]],
    index=['小明', '小红'],
    columns=['数学', '语文']
)

print("原表：")
print(df)




# stack：列转为内层索引
stacked = df.stack()
print("\nstack()：")
print(stacked)

# unstack：内层索引转为列
unstacked = stacked.unstack()
print("\nunstack()：")
print(unstacked)

dtype: int64

     数学  语文
小明  85  90
小红  78  92

df = pd.DataFrame({
    'A': ['a', 'a', 'b', 'b'],
    'B': ['x', 'y', 'x', 'y'],
    'C': [1, 2, 3, 4]
})

# 方法1：pivot_table
pt = df.pivot_table(index='A', columns='B', values='C', aggfunc='sum')

# 方法2：groupby + unstack
gb = df.groupby(['A', 'B'])['C'].sum().unstack()

print("pivot_table:")
print(pt)
print("\ngroupby + unstack:")
print(gb)


输出一致：

B  x  y
A
a  1  2
b  3  4

总结：何时使用哪种变形？

场景 推荐函数
长表 → 宽表（无重复） pivot
长表 → 宽表（有重复，需聚合） pivot_table
宽表 → 长表 melt / wide_to_long
列 ↔ 行索引转换 stack / unstack
频次统计表 crosstab
列表展开为行 explode
分类变量编码 get_dummies
