Conda的作用

python版本和第三方库众多,并且不同版本之间存在兼容性问题,Conda可以帮助我们管理python版本和依赖,避免依赖冲突 1.安装、管理、删除,更新python包 2.版本切换,不同的项目需要不同的python. 3.依赖隔离,比如项目A需要XXX 3.2版本,项目B需要XXX 4.5版本

Anaconda和Miniconda区别

特性 Anaconda Miniconda
发行公司 Anaconda, Inc. Anaconda, Inc.
安装方式 图形界面 + 命令行 命令行为主
默认包 集成 1500+ 科学计算常用包（NumPy、Pandas、Matplotlib、SciPy 等） 仅 Conda 本体 + Python 基础依赖
安装包大小 3~5 GB ~100 MB
环境灵活性 即装即用，适合初学者和数据科学用户 精简灵活，节省空间，可按需安装包
更新管理 包和环境更新方便，但占用空间大 包和环境可完全自定义，依赖需手动安装
使用场景 数据分析、机器学习、科研教学快速入门 节省空间、精简环境、服务器或脚本开发

配置源

1️⃣ 查看当前镜像源

在 cmd 或 Anaconda Prompt 中输入：

1
2
3
4
5
6
7

conda config --show channels

#这里可以确认你是否已经使用清华源。
channels:
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
  - defaults

2️⃣ 添加清华镜像源

执行以下命令：

1
2
3
4
5
6

conda config --add channels https://mirrors.cloud.tencent.com/anaconda/pkgs/free/
conda config --add channels https://mirrors.cloud.tencent.com/anaconda/pkgs/main/
conda config --add channels https://mirrors.cloud.tencent.com/anaconda/cloud/conda-forge/
conda config --add channels https://mirrors.cloud.tencent.com/anaconda/cloud/pytorch/
conda config --add channels https://mirrors.cloud.tencent.com/anaconda/pkgs/pro/
conda config --set show_channel_urls yes
  • --set show_channel_urls yes 会在安装包时显示具体源，方便确认。

⚠️ 添加镜像源后，最好清除索引缓存：

1

conda clean -i

然后可以创建新环境测试：

1

conda create -n myenv numpy

3️⃣ 删除指定镜像源

如果想移除某个镜像：

1

conda config --remove channels <镜像地址>

例如：

1

conda config --remove channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/

4️⃣ 恢复默认镜像源 (重置，只保留默认源)

1

conda config --remove-key channels

执行后，conda 会使用官方默认源。

## 环境管理（最常用）

创建环境	conda create -n xxx python=3.x
激活环境	conda activate xxx
安装包	conda install 包名
删除环境	conda env remove -n xxx --all
克隆环境（迁移）	conda create -n new --clone old
清理缓存	conda clean -a
查看所有环境	conda env list


## 包管理（最常用）
安装包
conda install 包名
指定 channel：
conda install -c conda-forge 包名

升级包
conda update 包名

搜索包
conda search 包名

升级 conda 本体：

conda update conda

卸载包
conda remove 包名

查看当前环境安装了哪些包
conda list

## 配置相关
查看配置
conda config --show

查看配置文件来源
conda config --show-sources

添加环境目录
conda config --add envs_dirs D:\conda_envs
    
设置默认频道
conda config --add channels conda-forge

清理缓存（非常常用）
conda clean -a

## 导出环境（备份/迁移）
导出 yml
conda env export > env.yaml

用 yml 创建环境
conda env create -f env.yaml
