02.ж•°жҚ®е®ҪеәҰ

2025-08-24
йҳ…иҜ»ж—¶й•ҝ: 5 еҲҶй’ҹ

д»Җд№ҲжҳҜж•°жҚ®е®ҪеәҰ

и®Ўз®—жңәдёӯеӯҳеӮЁзҡ„ж•°жҚ®жңҖз»ҲйғҪжҳҜдәҢиҝӣеҲ¶еӯҳеӮЁзҡ„пјҢи®Ўз®—жңәеҸӘи®ӨиҜҶ1е’Ң0пјҢдҪҶжҳҜдёҚжҳҜдёҖдёӢеӯҗе…ЁйғЁеӯҳиҝӣеҺ»зҡ„пјҢиҖҢжҳҜеҲҶжҲҗеҫҲеӨҡзҡ„е®№еҷЁпјҢжҜҸдёӘе®№еҷЁйғҪжңүжңҖеӨ§е®№йҮҸпјҢжҜ”еҰӮжңүзҡ„еҸҜд»ҘиЈ…4дёӘдәҢиҝӣеҲ¶ж•°пјҢжңүзҡ„еҸҜд»ҘиЈ…8дёӘдәҢиҝӣеҲ¶ж•°гҖӮжүҖд»ҘеңЁи®Ўз®—жңәдёӯпјҢз”ұдәҺеҸ—зЎ¬д»¶зҡ„еҲ¶зәҰпјҢж•°жҚ®йғҪжҳҜжңүй•ҝеәҰйҷҗеҲ¶зҡ„пјҲжҲ‘д»¬з§°дёәж•°жҚ®е®ҪеәҰпјү

еҰӮжһңиҰҒеӯҳеӮЁзҡ„ж•°и¶…иҝҮдәҶе®№еҷЁзҡ„ж•°жҚ®е®ҪеәҰи®Ўз®—жңәжҖҺд№ҲеӨ„зҗҶе‘ўпјҹзӣҙжҺҘе°Ҷй«ҳдҪҚеӨҡеҮәжқҘзҡ„дҪҚдёўејғпјҒ

жҜ”еҰӮдёҖдёӘе®№еҷЁзҡ„ж•°жҚ®е®ҪеәҰдёә4дҪҚпјҢйӮЈд№ҲеҰӮжһңжҲ‘зҺ°еңЁеӯҳдёҖдёӘж•°16пјҢеҢ–дёәдәҢиҝӣеҲ¶дёә10000пјҢз”ұдәҺеҸӘиғҪеӯҳ4дҪҚпјҢйӮЈд№ҲиҝӣдҪҚзҡ„еӨҡеҮәжқҘзҡ„дҪҚдјҡиў«дёўејғпјҢжңҖз»ҲиҜ»еҲ°зҡ„жҳҜ0000пјҢеҚі0

еӯҳеӮЁиҝҮзЁӢ

  • е®№еҷЁеҸӘжңү 4 дҪҚпјҢж„Ҹе‘ізқҖе®ғжң¬иҙЁе°ұжҳҜдёҖдёӘ 4 дҪҚеҜ„еӯҳеҷЁ/еӯҳеӮЁеҚ•е…ғгҖӮ
  • еҪ“дҪ жҠҠ 16пјҲдәҢиҝӣеҲ¶ 10000пјүеҶҷиҝӣеҺ»ж—¶пјҢи®Ўз®—жңәдјҡ еҸӘдҝқз•ҷдҪҺ 4 дҪҚпјҡ
  • 10000 пјҲ5 дҪҚеҺҹе§Ӣж•°жҚ®пјү
  • вҶ“ жҲӘж–ӯй«ҳдҪҚ
  • 0000 пјҲеҸӘеү© 4 дҪҚпјү
  • з»“жһңпјҡиҜ»еҮәжқҘжҳҜ 0

еҶҚзңӢеҮ дёӘдҫӢеӯҗ

  • еӯҳ 8пјҲдәҢиҝӣеҲ¶ 1000пјү вҶ’ 1000пјҲиҝҳиғҪж”ҫеҫ—дёӢпјү вҶ’ з»“жһң = 8
  • еӯҳ 15пјҲдәҢиҝӣеҲ¶ 1111пјү вҶ’ 1111 вҶ’ з»“жһң = 15
  • еӯҳ 16пјҲдәҢиҝӣеҲ¶ 10000пјү вҶ’ жҲӘж–ӯ вҶ’ 0000 вҶ’ з»“жһң = 0
  • еӯҳ 17пјҲдәҢиҝӣеҲ¶ 10001пјү вҶ’ жҲӘж–ӯ вҶ’ 0001 вҶ’ з»“жһң = 1
  • еӯҳ 18пјҲдәҢиҝӣеҲ¶ 10010пјү вҶ’ жҲӘж–ӯ вҶ’ 0010 вҶ’ з»“жһң = 2

ж— з¬ҰеҸ·ж•°гҖҒжңүз¬ҰеҸ·ж•°

д»ҘеүҚжҲ‘д»¬еҸӘжҳҜи®°дҪҸпјҡеҰӮжһңдёҖдёӘеӯҳеӮЁеҚ•е…ғеҸӘжңү 4 дҪҚпјҢ

  • ж— з¬ҰеҸ·ж•°пјҡ4 дҪҚе…ЁйғЁз”ЁжқҘиЎЁзӨәж•°еҖјпјҢеҸҜиЎЁзӨә 0000 еҲ° 1111пјҢжҖ»е…ұжңү 16 з§ҚеҸҜиғҪгҖӮ
  • жңүз¬ҰеҸ·ж•°пјҡжңҖй«ҳдҪҚдҪңдёәз¬ҰеҸ·дҪҚпјҢ0000 еҲ° 0111пјҲжңҖй«ҳдҪҚ 0пјүиЎЁзӨәжӯЈж•°пјҢ1000 еҲ° 1111пјҲжңҖй«ҳдҪҚ 1пјүиЎЁзӨәиҙҹж•°гҖӮ

дҪҶе®һйҷ…дёҠпјҢи®Ўз®—жңәе№¶дёҚвҖңи®ӨиҜҶвҖқжӯЈиҙҹеҸ·гҖӮеҺҹз ҒгҖҒеҸҚз ҒгҖҒиЎҘз ҒеҸӘжҳҜдәәзұ»дёәдәҶиЎЁзӨәжңүз¬ҰеҸ·ж•°е’Ңж–№дҫҝиҝҗз®—иҖҢеҲ¶е®ҡзҡ„и§„еҲҷпјҢзЎ¬д»¶еҸӘеӨ„зҗҶ 0 е’Ң 1пјҢе№¶дёҚдјҡзҗҶи§Јиҝҷдәӣж•°зҡ„жӯЈиҙҹж„Ҹд№үгҖӮ

4дҪҚе®ҪеәҰ(зҗҶи®әзӨәдҫӢ)

жіЁж„Ҹпјҡ1 дҪҚжҳҜзҗҶи®әдёҠзҡ„жңҖе°ҸеӯҳеӮЁеҚ•дҪҚпјҢдҪҶеңЁзҺ°д»Ј CPU е’ҢжұҮзј–е®һи·өдёӯпјҢжңҖе°Ҹзҡ„еҸҜж“ҚдҪңеҚ•дҪҚйҖҡеёёжҳҜ 8 дҪҚпјҲ1 еӯ—иҠӮпјүпјҢжүҖд»Ҙи®Ўз®—жңәдёӯжІЎжңү4дҪҚе®№еҷЁпјҢжҲ‘д»¬еҸӘжҳҜдёәдәҶеј•е…Ҙж–№дҫҝзҗҶи§ЈгҖӮ

8дҪҚе®ҪеәҰ

еҒҮи®ҫзҺ°еңЁ4дҪҚдёҚеӨҹиЎЁзӨәжҲ‘иҰҒеӯҳзҡ„ж•°дәҶпјҢжҲ‘жҚўдёҖдёӘеӨ§дёҖзӮ№зҡ„е®№еҷЁпјҢеӯҳеӮЁж•°жҚ®зҡ„иҢғеӣҙдёә0x00еҲ°0xFFпјҢеҚіж•°жҚ®е®ҪеәҰдёә8дҪҚпјҢйӮЈд№ҲжӯӨж—¶е®№еҷЁеҰӮдҪ•иЎЁзӨә

1. ж— з¬ҰеҸ·ж•°иЎЁзӨә

еҜ№дәҺ 8дҪҚж— з¬ҰеҸ·ж•°пјҢиҢғеӣҙд»Һ 0x00 еҲ° 0xFFпјҢд№ҹе°ұжҳҜд»Һ 0 еҲ° 255пјҢе…ұжңү 256 дёӘеҸҜиғҪзҡ„ж•°еҖјгҖӮиҝҷжҳҜеӣ дёәжҜҸдёӘдҪҚзҪ®дёҠеҸҜд»ҘжҳҜ 0 жҲ– 1пјҢжүҖд»Ҙжңү 2^8 = 256 з§ҚдёҚеҗҢзҡ„з»„еҗҲгҖӮ

  • жңҖе°ҸеҖјпјҡ0x00пјҢеҚі 0гҖӮ
  • жңҖеӨ§еҖјпјҡ0xFFпјҢеҚі 255гҖӮ

2. жңүз¬ҰеҸ·ж•°иЎЁзӨә

еҜ№дәҺ 8дҪҚжңүз¬ҰеҸ·ж•°пјҢжҲ‘д»¬йңҖиҰҒдҪҝз”Ё иЎҘз ҒиЎЁзӨәжі•гҖӮдёәдәҶиЎЁзӨәжӯЈиҙҹж•°пјҢйҖҡеёёзәҰе®ҡ жңҖй«ҳдҪҚпјҲеҚі з¬ҰеҸ·дҪҚпјүдёә 0 ж—¶иЎЁзӨәжӯЈж•°пјҢз¬ҰеҸ·дҪҚдёә 1 ж—¶иЎЁзӨәиҙҹж•°гҖӮж №жҚ®иЎҘз Ғзҡ„и§„еҲҷпјҢиҢғеӣҙиў«еҲҶдёәдёӨеҚҠпјҡ

жӯЈж•°иҢғеӣҙпјҡ0x00 еҲ° 0x7FпјҢеҚід»Һ 0 еҲ° 127пјҢиҝҷйғЁеҲҶж•°жҚ®з”ЁиЎҘз ҒиЎЁзӨәж—¶пјҢз¬ҰеҸ·дҪҚдёә 0пјҢе…¶дҪҷ 7 дҪҚиЎЁзӨәжӯЈж•°гҖӮ

  • 0x00 = 00000000пјҢиЎЁзӨәжӯЈж•° 0гҖӮ

  • 0x7F = 01111111пјҢиЎЁзӨәжӯЈж•° 127пјҲз¬ҰеҸ·дҪҚдёә 0пјүгҖӮ

иҙҹж•°иҢғеӣҙпјҡ0x80 еҲ° 0xFFпјҢеҚід»Һ -128 еҲ° -1пјҢз¬ҰеҸ·дҪҚдёә 1пјҢеү©дҪҷйғЁеҲҶиЎЁзӨәиҙҹж•°гҖӮиҙҹж•°зҡ„иЎҘз ҒжҳҜйҖҡиҝҮеҜ№жӯЈж•°еҸ–еҸҚеҠ  1 жқҘеҫ—еҲ°зҡ„гҖӮ

  • 0x80 = 10000000пјҢиЎЁзӨәиҙҹж•° -128пјҲз¬ҰеҸ·дҪҚдёә 1пјүгҖӮ

  • 0xFF = 11111111пјҢиЎЁзӨәиҙҹж•° -1гҖӮеҚі11111111 вҶ’ еҸ–еҸҚ 00000000 вҶ’ +1 вҶ’ 00000000 вҶ’ -1

16дҪҚе®ҪеәҰ

16дҪҚж•°жҚ®е®ҪеәҰе®№еҷЁеӯҳеӮЁж— з¬ҰеҸ·ж•°иҢғеӣҙжҳҜд»Һ0еҲ°FFFF еӯҳеӮЁжңүз¬ҰеҸ·ж•°д»Һ0еҲ°7FFFиЎЁзӨәжӯЈж•°пјҢд»Һ8000еҲ°FFFFиЎЁзӨәиҙҹж•°

32дҪҚе®ҪеәҰ

зӣ®еүҚи®Ўз®—жңәдёӯжңҖеёёи§Ғзҡ„д№ҹжҳҜиҝҷдёӘж•°жҚ®е®ҪеәҰзҡ„е®№еҷЁ

32дҪҚж•°жҚ®е®ҪеәҰе®№еҷЁеӯҳеӮЁж— з¬ҰеҸ·ж•°иҢғеӣҙжҳҜд»Һ0еҲ°FFFFFFFF еӯҳеӮЁжңүз¬ҰеҸ·ж•°д»Һ0еҲ°7FFFFFFFиЎЁзӨәжӯЈж•°пјҢд»Һ80000000еҲ°FFFFFFFFиЎЁзӨәиҙҹж•°
