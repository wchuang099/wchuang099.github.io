PySimpleGUI 学习指南

什么是 PySimpleGUI

PySimpleGUI 是一个封装了 Tkinter（默认）、Qt、WxPython、Remi 的 简易 GUI 框架。 它的目标：用最少的代码，快速写出桌面应用 GUI。 特点：语法直观、学习曲线低，适合初学者和快速原型开发。

PySimpleGUI 的三大核心

  • 布局（Layout） 界面由控件（如按钮、文本、输入框）组成，控件排布在一个二维列表中。 每一行是一个列表，整个窗口是一个大的列表。

  • 窗口（Window） 用 sg.Window 创建窗口。 接收标题、布局等参数。

  • 事件循环（Event Loop） GUI 需要不断等待用户操作（事件），比如点击按钮、输入文本。 用 window.read() 获取 event 和 values。 在 PySimpleGUI 中，event 的值通常就是你为控件设置的 key。 没有指定 key 参数，那么它的事件标识（event）就是按钮上显示的文字内容。

PySimpleGUI 常用控件速查表

一、基础元素

控件类型 代码示例 说明
文本 sg.Text("显示文本", key="text") 显示静态文字；不支持 text_container 参数
输入框 sg.Input(key="input", size=(20, 1)) 单行文本输入
多行输入 sg.Multiline(size=(40, 5), key="multiline", disabled=True) 多行文本，可设为只读
按钮 sg.Button("确定", key="submit") 点击触发事件
图片 sg.Image("logo.png", key="image") 支持 PNG/JPG 等格式

二、选择类控件

控件类型 代码示例 说明
复选框 sg.Checkbox("同意协议", key="chk_agree") 返回 True/False
单选框 sg.Radio("男", "gender", key="radio_male") 同一组名（如 "gender"）互斥
下拉框 sg.Combo(["A", "B", "C"], key="combo_selection") 可编辑或只读（加 readonly=True）
列表框 sg.Listbox(values=["苹果", "香蕉"], size=(20, 3), key="list_items") 返回选中项列表
滑块 sg.Slider(range=(0, 100), key="slider_value", orientation='h') 水平/垂直滑块

三、文件与文件夹

功能 代码示例 说明
选择文件 sg.FileBrowse(key="file_browse") 需配合 Input 使用
保存文件 sg.FileSaveAs(key="file_saveas") 弹出“另存为”对话框
选择文件夹 sg.FolderBrowse(key="folder_browse") 返回目录路径
输入+选择 [sg.Input(key="file_path"), sg.FileBrowse(target="file_path")] target 指定输入框 key

FileBrowse 默认将路径填入前一个 Input，用 target 可指定任意 key。

四、进度与状态

控件类型 代码示例 说明
进度条 sg.ProgressBar(100, orientation='h', size=(20, 20), key="pbar") 更新：window["pbar"].update(current_value)
状态栏 sg.Text("就绪", text_color="gray", key="status") PySimpleGUI 无原生 StatusBar，常用 Text 模拟

五、容器布局（核心！）

容器类型 代码示例 说明
列容器 sg.Column([[sg.Button("A")]], scrollable=True, vertical_scroll_only=True) 控制布局的核心：可设固定宽高、滚动、隐藏
框架 sg.Frame("测试",[[sg.Button("a")]],pad=(10, 10)) 视觉分组，带标题边框；可加 size= 强制尺寸
页签 sg.Tab("首页", [[sg.Text("内容")]]) 单个标签页
页签组 sg.TabGroup([[tab1, tab2]]) 包含多个 Tab
#Column 是实现复杂布局（如可折叠面板、滚动区域）的关键；Frame 只是装饰性分组。

六、菜单与底部信息

功能 代码示例 说明
顶部菜单 sg.Menu([['文件', ['退出']], ['帮助', ['关于']]]) 菜单项点击后，event 为菜单文字
底部信息（左+右对齐） [sg.Text("左"), sg.Push(), sg.Text("右")] ` Push = 弹簧，撑开左右
水平分隔线 sg.HSeparator() 视觉分割区域
垂直分隔线 sg.VSeperator() 分隔左右区域

七、弹窗（Popup）

弹窗类型 代码示例 返回值 / 说明
普通提示 sg.popup("消息") 返回 "OK"，阻塞程序直到关闭
是/否选择 sg.popup_yes_no("确定吗?") 返回 "Yes" 或 "No"
确定/取消 sg.popup_ok_cancel("继续吗?") 返回 "OK" 或 "Cancel"
文件选择 sg.popup_get_file("选择文件") 返回路径字符串或 None
错误提示 sg.popup_error("出错了!") 红色背景，强调错误
可滚动文本 sg.popup_scrolled("长文本...") 自动添加滚动条

所有的 sg.popup* 系列函数都不接受 key 参数。不参与事件循环

    title="窗口标题",
    layout=your_layout,      # 必填：界面布局

    finalize=True,           # 可选：立刻把窗口和所有控件真正创建出来（用于初始化后立即操作控件）

    size=(600, 400),    # 初始尺寸 (宽, 高)
    icon="" 
)

window 是通过 sg.Window(…) 创建的窗口实例，是与界面交互的核心对象。几乎所有动态操作都通过它完成。

2.读取事件与值（主循环）

1

event, values = window.read()

•event：触发的事件（如按钮文字、key、sg.WIN_CLOSED） •values：字典，包含所有带 key 的输入控件的当前值 •values["-INPUT-"] # 获取 Input 内容 •values["-CHECK-"] # 获取 Checkbox 状态 (True/False) window.read() 是阻塞调用，程序会停在这里等待用户操作。

3.更新控件内容

1

window["key"].update(new_value, **kwargs)

适用于： •Text：更新显示文字 •Input / Multiline：设置文本 •Button：改文字、禁用等 •Image：换图 •ProgressBar：更新进度 示例：

1
2
3

window["-STATUS-"].update("处理中...", text_color="blue")
window["-PROGRESS-"].update(50)  # 进度条到 50%
window["-SUBMIT-"].update(disabled=True)  # 禁用按钮

4.控制控件可见性

window[“key”].update(visible=False) # 隐藏 window[“key”].update(visible=True) # 显示 常用于动态面板切换（配合 Column 或 Frame）。

5.聚焦与清空

window["-INPUT-"].set_focus() # 光标聚焦到输入框 window["-INPUT-"].update("") # 清空内容 6.关闭窗口

window.close() •释放资源 •必须在程序结束前调用，否则可能卡住或报错

window = sg.Window("标题", layout)
while True:
    event, values = window.read()
    if event in (sg.WIN_CLOSED, "退出"):  # 关闭窗口或点击退出
        break
    elif event == "关于":
        sg.popup("关于信息")
    elif event == "执行":
        # 处理逻辑
        window["output"].update("完成！")
window.close()

import PySimpleGUI as sg

# ==============================
# 全局登记本：按钮 → 对应函数
# ==============================
event_handlers = {}

def register_event(event_name):
    """装饰器：登记事件处理函数"""
    def decorator(func):
        event_handlers[event_name] = func
        return func
    return decorator


# ==============================
# 功能函数：加减乘除
# ==============================
@register_event("加法")
def add(values):
    try:
        a, b = float(values["num1"]), float(values["num2"])
        sg.popup(f"结果: {a} + {b} = {a + b}")
    except ValueError:
        sg.popup_error("请输入数字！")

@register_event("减法")
def sub(values):
    try:
        a, b = float(values["num1"]), float(values["num2"])
        sg.popup(f"结果: {a} - {b} = {a - b}")
    except ValueError:
        sg.popup_error("请输入数字！")

@register_event("乘法")
def mul(values):
    try:
        a, b = float(values["num1"]), float(values["num2"])
        sg.popup(f"结果: {a} × {b} = {a * b}")
    except ValueError:
        sg.popup_error("请输入数字！")

@register_event("除法")
def div(values):
    try:
        a, b = float(values["num1"]), float(values["num2"])
        if b == 0:
            sg.popup_error("除数不能为 0！")
        else:
            sg.popup(f"结果: {a} ÷ {b} = {a / b}")
    except ValueError:
        sg.popup_error("请输入数字！")


# ==============================
# GUI 布局
# ==============================
layout = [
    [sg.Text("数字1:"), sg.Input(key="num1")],
    [sg.Text("数字2:"), sg.Input(key="num2")],
    [sg.Button("加法"), sg.Button("减法"), sg.Button("乘法"), sg.Button("除法")],
    [sg.Button("退出")]
]

window = sg.Window("装饰器计算器", layout)

# ==============================
# 事件循环
# ==============================
while True:
    event, values = window.read()
    if event in (sg.WIN_CLOSED, "退出"):
        break

    handler = event_handlers.get(event)
    if handler:
        handler(values)

window.close()
