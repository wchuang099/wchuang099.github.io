什么是 PySimpleGUI

PySimpleGUI 是一个封装了 Tkinter（默认）、Qt、WxPython、Remi 的 简易 GUI 框架。 它的目标：用最少的代码，快速写出桌面应用 GUI。 特点：语法直观、学习曲线低，适合初学者和快速原型开发。

PySimpleGUI 的三大核心

  • 布局（Layout） 界面由控件（如按钮、文本、输入框）组成，控件排布在一个二维列表中。 每一行是一个列表，整个窗口是一个大的列表。

  • 窗口（Window） 用 sg.Window 创建窗口。 接收标题、布局等参数。

  • 事件循环（Event Loop） GUI 需要不断等待用户操作（事件），比如点击按钮、输入文本。 用 window.read() 获取 event 和 values。 在 PySimpleGUI 中，event 的值通常就是你为控件设置的 key。

PySimpleGUI 常用控件速查表

基础元素

控件类型 代码示例 说明
文本 sg.Text("显示文本", key="text") 显示一段静态文字
输入框 sg.Input(key="input") 单行文本输入
多行输入 sg.Multiline(size=(40, 5), key="multiline") 多行文本输入，支持滚动
按钮 sg.Button("确定", key="submit") 点击触发事件
图片 sg.Image("logo.png", key="image") 显示本地图片文件

选择类控件

控件类型 代码示例 说明
复选框 sg.Checkbox("同意协议", key="chk_agree") 可多选，返回布尔值
单选框 sg.Radio("男", "gender", key="radio_male") 同一组内互斥，"gender" 为组名
下拉框 sg.Combo(["A", "B", "C"], key="combo_selection") 下拉选择，返回选中项
列表框 sg.Listbox(values=["苹果", "香蕉"], size=(20, 3), key="list_items") 可多选或单选，返回列表
滑块 sg.Slider(range=(0, 100), key="slider_value") 拖动选择数值

文件与文件夹

功能 代码示例 说明
选择文件 sg.FileBrowse(key="file_browse") 打开文件选择对话框
保存文件 sg.FileSaveAs(key="file_saveas") 打开“另存为”对话框
选择文件夹 sg.FolderBrowse(key="folder_browse") 选择目录路径
输入+选择 [sg.Input(key="file_input"), sg.FileBrowse(key="file_browse_btn")] 组合：输入框 + 文件选择按钮

进度与状态

控件类型 代码示例 说明
进度条 sg.ProgressBar(max_value=100, orientation="h", size=(20, 20), key="progress_bar") 显示任务进度，需配合更新
状态栏 sg.StatusBar("就绪", size=(40, 1), key="status_bar") 通常放在窗口底部，显示状态信息

容器布局

容器类型 代码示例 说明
框架 sg.Frame("设置", [[sg.Text("内容", key="frame_text")]], key="frame_settings") 将控件分组，带标题边框
页签 sg.Tab("第一页", [[sg.Text("内容1", key="tab1_text")]], key="tab1") 单个页签内容容器
页签组 sg.TabGroup([[tab1, tab2]], key="tab_group") 包含多个 Tab 的容器，实现多页界面

弹窗（Popup）

弹窗类型 代码示例 返回值 / 说明
普通提示 sg.popup("消息") 显示消息，点击“确定”关闭
是/否选择 sg.popup_yes_no("确定吗?") 返回 "Yes" 或 "No"
确定/取消 sg.popup_ok_cancel("继续吗?") 返回 "OK" 或 "Cancel"
文件选择 sg.popup_get_file("选择文件") 返回文件路径，或 None
错误提示 sg.popup_error("错误!") 红色主题，突出显示错误
可滚动文本 sg.popup_scrolled("很多很多文字...") 内容过多时可滚动查看

import PySimpleGUI as sg

# 1. 定义布局
layout = [
    [sg.Text("请输入你的名字:")],
    [sg.Input(key="name")],
    [sg.Button("确定"), sg.Button("退出")]
]

# 2. 创建窗口
window = sg.Window("Hello PySimpleGUI", layout)

# 3. 事件循环
while True:
    event, values = window.read()  # 事件, 输入值字典
    if event == sg.WIN_CLOSED or event == "退出":
        break
    if event == "确定":
        sg.popup(f"你好, {values['name']}!")

# 4. 关闭窗口
window.close()

👉 运行效果：

输入名字，点“确定”，会弹出提示框。

点“退出”或关闭窗口，就结束程序。

import PySimpleGUI as sg

layout = [
    [sg.Text("用户名:"), sg.Input(key="username")],
    [sg.Text("密码:"), sg.Input(key="password", password_char="*")],
    [sg.Button("登录"), sg.Button("取消")]
]

window = sg.Window("登录系统", layout)

while True:
    event, values = window.read()
    if event in (sg.WIN_CLOSED, "取消"):
        break
    if event == "登录":
        if values["username"] == "admin" and values["password"] == "123":
            sg.popup("登录成功!")
        else:
            sg.popup("用户名或密码错误!")

window.close()

## 简单记事本
```python
import PySimpleGUI as sg

layout = [
    [sg.Multiline(size=(40, 10), key="text")],
    [sg.Button("保存"), sg.Button("退出")]
]

window = sg.Window("记事本", layout)

while True:
    event, values = window.read()
    if event in (sg.WIN_CLOSED, "退出"):
        break
    if event == "保存":
        with open("note.txt", "w", encoding="utf-8") as f:
            f.write(values["text"])
        sg.popup("保存成功!")

window.close()

import PySimpleGUI as sg

# ==============================
# 全局登记本：按钮 → 对应函数
# ==============================
event_handlers = {}

def register_event(event_name):
    """装饰器：登记事件处理函数"""
    def decorator(func):
        event_handlers[event_name] = func
        return func
    return decorator


# ==============================
# 功能函数：加减乘除
# ==============================
@register_event("加法")
def add(values):
    try:
        a, b = float(values["num1"]), float(values["num2"])
        sg.popup(f"结果: {a} + {b} = {a + b}")
    except ValueError:
        sg.popup_error("请输入数字！")

@register_event("减法")
def sub(values):
    try:
        a, b = float(values["num1"]), float(values["num2"])
        sg.popup(f"结果: {a} - {b} = {a - b}")
    except ValueError:
        sg.popup_error("请输入数字！")

@register_event("乘法")
def mul(values):
    try:
        a, b = float(values["num1"]), float(values["num2"])
        sg.popup(f"结果: {a} × {b} = {a * b}")
    except ValueError:
        sg.popup_error("请输入数字！")

@register_event("除法")
def div(values):
    try:
        a, b = float(values["num1"]), float(values["num2"])
        if b == 0:
            sg.popup_error("除数不能为 0！")
        else:
            sg.popup(f"结果: {a} ÷ {b} = {a / b}")
    except ValueError:
        sg.popup_error("请输入数字！")


# ==============================
# GUI 布局
# ==============================
layout = [
    [sg.Text("数字1:"), sg.Input(key="num1")],
    [sg.Text("数字2:"), sg.Input(key="num2")],
    [sg.Button("加法"), sg.Button("减法"), sg.Button("乘法"), sg.Button("除法")],
    [sg.Button("退出")]
]

window = sg.Window("装饰器计算器", layout)

# ==============================
# 事件循环
# ==============================
while True:
    event, values = window.read()
    if event in (sg.WIN_CLOSED, "退出"):
        break

    handler = event_handlers.get(event)
    if handler:
        handler(values)

window.close()
