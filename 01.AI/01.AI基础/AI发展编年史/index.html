    人工智能（AI）发展编年史#

    基于 aibook.ren 整理简化。内容经过精简重述，供快速了解 AI 发展脉络。

    总览：七个阶段#

    诞生（1943-1956）→ 黄金年代（1956-1974）→ 第一次低谷（1974-1980）
→ 繁荣期（1980-1987）→ 第二次低谷（1987-1993）→ 平稳过渡（1993-2011）
→ 大模型时代（2017-至今）

    第一阶段：AI 的诞生（1943-1956）#

    年份 事件
    1943 图灵团队研制"巨人"机器破解德军密码，被视为计算机与 AI 科学的先驱
    1946 全球第一台通用计算机 ENIAC 诞生，为 AI 研究提供物质基础
    1950 马文·明斯基建造世界第一台神经网络计算机；图灵提出"图灵测试"
    1956 约翰·麦卡锡提出"人工智能"一词，AI 正式诞生。与明斯基共建 MIT AI 实验室

    第二阶段：黄金年代（1956-1974）#

    年份 事件
    1958 美国国防高级研究计划署（DARPA）大规模投资 AI；麦卡锡开发 LISP 语言
    1959 首台工业机器人诞生；西洋跳棋程序战胜人类大师
    1964 首台聊天机器人 ELIZA 诞生（MIT）
    1965 首个专家系统 DENDRAL 亮相，能分析化学分子结构
    1968 首台 AI 机器人 Shakey 诞生，能自主感知环境并执行任务
    1970 人机对话系统 SHRDLU 诞生，能理解语义并操作虚拟方块

    第三阶段：第一次低谷（1974-1980）#

    年份 事件
    1973 拉特希尔报告严厉批评 AI 研究"完全失败"，各国大幅削减 AI 经费
    1976 医疗专家系统 MYCIN 发布，但整体 AI 进入低谷
    — 瓶颈：计算机性能不足、处理复杂问题能力有限、数据严重缺失

    第四阶段：繁荣期（1980-1987）#

    年份 事件
    1980 专家系统 XCON 为公司年省超 4000 万美元，“知识处理"成为主流
    1981 日本投资 8.5 亿美元启动"第五代计算机"项目，各国纷纷响应
    1982 Hopfield 网络问世；反向传播算法（Backpropagation）被推广，联结主义复兴
    1986 《分布式并行处理》论文集出版，神经网络领域获得统一和促进

    第五阶段：第二次低谷（1987-1993）#

    年份 事件
    1987 PC 性能超过专用 AI 硬件，5 亿美元的 AI 硬件产业崩溃
    1991 日本"第五代工程"宣告失败
    1993 学界反思：自上而下的规则编程路线受到质疑，“从下而上"的学习思想开始萌芽

    第六阶段：平稳过渡（1993-2011）#

    年份 事件
    1997 IBM"深蓝"战胜国际象棋世界冠军卡斯帕罗夫
    1997 LSTM（长短期记忆网络）被提出，至今仍用于语音和手写识别
    2004 杰夫·霍金斯出版《人工智能的未来》，提出大脑记忆预测理论
    2006 杰弗里·辛顿奠定深度学习核心架构，至今仍是 AI 基石
    2007 李飞飞发起 ImageNet 项目（1400 万张图片、2 万+类别）
    2011 IBM Watson 参加智力问答节目战胜人类冠军
    2012 吴恩达团队用 GPU 实现大规模无监督学习，机器自主识别猫图——里程碑事件
    2012 深度卷积神经网络在 ImageNet 挑战赛夺冠，深度学习革命正式开始
    2014 谷歌成为首个通过美国自驾车测试的公司
    2016-17 AlphaGo 战胜围棋冠军；AlphaGo Zero 从零自学 3 天横扫旧版本

    第七阶段：大模型时代（2017-至今）#

    2017：Transformer 诞生#

    Google 发表《Attention is All You Need》，Transformer 架构成为后续所有大模型的核心，引发 AI 革命。

    2020-2021：OpenAI 崛起，AIGC 拉开序幕#

    时间 事件
    2020 OpenAI 发布 GPT-3（1750 亿参数）、开放 API
    2021 OpenAI 发布 CLIP（文本-图像连接）和 DALL·E（文本生成图像）

    2022：ChatGPT 引爆全球#

    时间 事件
    4月 DALL·E 2 升级，文生图进入实用阶段
    8月 AI 画作《太空歌剧院》在人类美术比赛获第一名
    9月 DreamFusion（文本→3D）、Make-A-Video（文本→视频）发布
    11月30日 ChatGPT 发布，引入 RLHF 技术，两个月用户破 1 亿

    2023：百模大战（大模型元年）#

    时间 关键事件
    3月 Meta 开源 LLaMA；OpenAI 发布 GPT-4（多模态）；百度发布文心一言；清华发布 ChatGLM
    4月 阿里发布通义千问；AutoGPT 爆火（GitHub 10万+ star）
    5月 科大讯飞发布星火大模型；Google 发布 PaLM 2
    7月 Meta 发布 Llama 2（支持商用）；大模型进入开源商用阶段
    9月 微软 Copilot 嵌入 Windows 11；Mistral 7B 发布；腾讯发布混元大模型
    10月 百度文心 4.0 发布；智谱发布 ChatGLM3（支持 Agent）
    11月 OpenAI 发布 GPT-4 Turbo；阿里通义千问 Qwen-72B 开源
    12月 Mamba 架构发布（挑战 Transformer）；Google 发布 Gemini

    2024：大模型商业化元年#

    时间 关键事件
    2月 欧盟通过《人工智能法案》；OpenAI 发布 Sora（文生视频）
    3月 Anthropic 发布 Claude 3（首次全面超越 GPT-4）；全球首位 AI 软件工程师 Devin 发布
    6月 字节跳动发布"豆包"大模型，推理成本仅为行业 1/100，价格战打响
    7月 特斯拉发布 FSD 4.0 自动驾驶芯片
    9月 联合国发布《全球 AI 伦理指南》
    10月 Google DeepMind 发布 AlphaFold 3（蛋白质预测准确率 95%）
    11月 Anthropic 提出 MCP（模型上下文协议），统一 AI 与外部工具的交互标准

    2025：AI Agent 元年#

    时间 关键事件
    1月 DeepSeek-R1 发布，极低成本对标顶尖性能，震动全球
    2月 Google 开放 Gemini 1.5 Pro 视频理解能力；Meta 开源 Code Llama 70B
    3月 MCP 联盟成立（Google、Meta、阿里云等），推动 Agent 标准化
    4月 阿里开源 Qwen-110B + Qwen-Agent + Qwen-MCP，首个支持 MCP 的中文开源生态
    5月 Google 确立"AI-First"战略；微软推出 Copilot+ PC
    6月 Agent 即服务（AaaS）成为新范式，云厂商提供智能体托管平台
    10月 通义千问发布 Qwen-Edge 系列，端侧 Agent 规模化部署
    11月 OpenClaw 开源发布，迅速成为 GitHub 最火 AI 项目；Claude 4.5 发布
    12月 Google Gemini 3 发布（千万级 Token 上下文）；DeepSeek V3.2 发布

    2026（至今）#

    时间 关键事件
    1月 NVIDIA 发布 Vera Rubin 架构（推理与自主决策导向）；OpenClaw 突破 18 万 GitHub star
    2月 字节发布豆包 2.0；春节"AI 红包大战"爆发，AI 从工具演变为"生活环境”
    2月 OpenClaw 误删事件引发全社会对"Agent 权限管控"的讨论

    AI 发展的关键转折点#

    转折点 意义
    1956 “人工智能"命名 AI 作为学科正式诞生
    2006 辛顿深度学习 奠定现代 AI 技术基础
    2012 ImageNet 深度学习夺冠 深度学习革命开始
    2017 Transformer 论文 大模型时代的技术基石
    2022 ChatGPT 发布 AI 走向大众，全球 AI 热潮
    2025 Agent + MCP AI 从"对话"走向"自主行动”

    AI 发展规律#

    1. 周期性起伏：AI 经历了多次"热潮→低谷→再热潮"的循环，每次低谷都是因为期望过高而技术跟不上
    2. 算力驱动：每次突破都伴随着算力的飞跃（ENIAC → GPU → TPU → NPU）
    3. 数据为王：从 ImageNet 到互联网语料，数据规模的增长是模型进步的关键
    4. 从规则到学习：AI 的方法论从"人写规则"转向"机器自己学”，这是最根本的范式转变
    5. 从对话到行动：2025 年起，AI 从"能说会道"进化为"能做事"的 Agent 时代
