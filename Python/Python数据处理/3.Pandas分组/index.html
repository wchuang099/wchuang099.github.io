    分组模式及其对象#

    分组的一般模式#

    概念： Pandas 分组通常遵循三个步骤：

    1. 分割（split）：根据某些条件把数据分组。
    2. 应用（apply）：对每个分组进行计算。
    3. 合并（combine）：将分组结果合并成一个新的数据结构。 df.groupby(分组依据)[数据来源].使用操作()
    import pandas as pd
# 创建示例数据
data = {
    '城市': ['北京', '上海', '北京', '广州', '上海', '广州'],
    '年份': [2020, 2020, 2021, 2020, 2021, 2021],
    '销售额': [100, 200, 150, 300, 250, 350]
}

df = pd.DataFrame(data)
print(df)
# 分组模式：按城市分组，计算销售额总和
grouped = df.groupby('城市')['销售额'].sum()
print(grouped)

    分组依据的本质#

    概念： 分组依据可以是：

    • 某一列（或多列）
    • Series、数组、字典
    • 函数（如 lambda x: x.year）
    # 按年份分组
grouped_year = df.groupby(df['年份'])['销售额'].sum()
print(grouped_year)

# 按自定义函数分组（销售额>200为一组，小于等于200为另一组）
grouped_custom = df.groupby(lambda x: '大额' if df.loc[x, '销售额'] > 200 else '小额')['销售额'].sum()
print(grouped_custom)

    GroupBy对象#

    概念： groupby 返回一个 GroupBy 对象，它本身不会立即计算，而是惰性计算。 可以用 groups 查看分组索引。

    测试代码：

    grouped_obj = df.groupby('城市')
print(type(grouped_obj))
print(grouped_obj.groups)

    分组的三大操作#

    概念：

    1. 聚合（Aggregation）：对每组计算汇总值。
    2. 变换（Transformation）：对每组计算一个与原数据同维度的值。
    3. 过滤（Filtering）：根据某组条件保留或删除分组。
    # 聚合
print(grouped_obj['销售额'].sum())
# 变换
print(grouped_obj['销售额'].transform('mean'))
# 过滤
print(grouped_obj.filter(lambda x: x['销售额'].sum() > 400))

    聚合函数#

    内置聚合函数#

    概念： 常用函数： sum(), mean(), max(), min(), count()

    print(grouped_obj['销售额'].mean())
print(grouped_obj['销售额'].max())
print(grouped_obj['销售额'].count())

    agg方法#

    概念： agg 可以同时使用多个聚合函数，也可以自定义函数。

    # 多个聚合函数
print(grouped_obj['销售额'].agg(['sum', 'mean', 'max']))
# 自定义函数
print(grouped_obj['销售额'].agg(lambda x: x.max() - x.min()))

    变换和过滤#

    变换函数与 transform 方法#

    概念： transform 保持原数据形状，对每个组执行函数。 适合用于归一化、填充缺失值等。

    # 标准化（每组减去均值再除以标准差）
df['销售额标准化'] = grouped_obj['销售额'].transform(lambda x: (x - x.mean()) / x.std())
print(df)

    组索引与过滤#

    概念： filter 保留满足条件的组 条件函数返回 True/False

    # 过滤总销售额大于400的组
filtered_df = grouped_obj.filter(lambda x: x['销售额'].sum() > 400)
print(filtered_df)

    跨列分组#

    apply的引入#

    概念： apply 可以对每组进行任意操作，返回结果可以是不同形状。 比 agg 更灵活。

    # 每组增加一列，显示销售额是否高于组均值
def flag_high_sales(group):
    group['高于均值'] = group['销售额'] > group['销售额'].mean()
    return group

df_applied = grouped_obj.apply(flag_high_sales)
print(df_applied)

    apply的使用#

    概念： 可以对每组返回单值、多值、甚至 DataFrame 支持复杂计算

    # 每组取销售额最大的记录
max_sales = grouped_obj.apply(lambda x: x[x['销售额'] == x['销售额'].max()])
print(max_sales)
