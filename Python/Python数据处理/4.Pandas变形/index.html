    长宽表的变形#

    pivot将长表转为宽表（单个聚合函数）#

    功能：将一列作为新列名，一列作为新行索引，填充值来自另一列。
    限制：索引和列的组合必须唯一（不能有重复）。

    语法：df.pivot(index='行索引', columns='列名', values='值')

    案例：学生成绩表（长 → 宽）

    import pandas as pd

# 原始长表
df_long = pd.DataFrame({
    '学生': ['小明', '小明', '小红', '小红'],
    '科目': ['数学', '语文', '数学', '语文'],
    '成绩': [85, 90, 78, 92]
})

print("原始长表：")
print(df_long)

# 转为宽表
df_wide = df_long.pivot(index='学生', columns='科目', values='成绩')
print("\npivot 转宽表：")
print(df_wide)

#输出
原始长表：
   学生  科目  成绩
0  小明  数学  85
1  小明  语文  90
2  小红  数学  78
3  小红  语文  92

pivot 转宽表：
科目  数学  语文
学生
小明   85   90
小红   78   92

    pivot_table支持聚合的宽表转换（pivot 的增强版）#

    功能：支持聚合（如 mean, sum），可处理重复索引。

    语法:df.pivot_table(index='行',columns='列', values='值', aggfunc='mean') 案例：带重复数据的成绩表

    # 含重复数据（小明考了两次数学）

df_repeat = pd.DataFrame({
    '学生': ['小明', '小明', '小明', '小红'],
    '科目': ['数学', '数学', '语文', '数学'],
    '成绩': [85, 88, 90, 78]
})

# 使用 pivot_table 计算平均分

df_pivot_table = df_repeat.pivot_table(
    index='学生',
    columns='科目',
    values='成绩',
    aggfunc='mean'
)

print("pivot_table（平均分）：")
print(df_pivot_table)

#输出
科目     数学   语文
学生
小明  86.5   90
小红  78.0  NaN

    melt将宽表转为长表#

    功能：将多列合并为两列（变量名 + 值），是 pivot 的逆操作。

    语法:df.melt(id_vars='不变列', value_vars='要合并的列', var_name='变量名', value_name='值')

    案例：宽表转长表

    # 宽表
df_wide = pd.DataFrame({
    '学生': ['小明', '小红'],
    '数学': [85, 78],
    '语文': [90, 92]
})

# 转为长表
df_melt = df_wide.melt(
    id_vars='学生',
    value_vars=['数学', '语文'],
    var_name='科目',
    value_name='成绩'
)

print("melt 转长表：")
print(df_melt)


输出：


   学生  科目  成绩
0  小明  数学  85
1  小红  数学  78
2  小明  语文  90
3  小红  语文  92

    wide_to_long处理复杂列名的宽转长#

    功能：适用于列名有规律（如 score_math, score_chinese）的宽表。

    案例：带前缀的宽表

    df_complex = pd.DataFrame({
    '学生': ['小明', '小红'],
    'score_math': [85, 78],
    'score_chinese': [90, 92],
    'time_math': ['2023', '2023'],
    'time_chinese': ['2023', '2023']
})

# 转换
df_long = pd.wide_to_long(
    df_complex,
    stubnames=['score', 'time'],
    i='学生',
    j='科目',
    sep='_',
    suffix='\w+'
)

print("wide_to_long 结果：")
print(df_long)

输出：
              score   time
学生  科目
小明  math       85   2023
小红  math       78   2023
小明  chinese    90   2023
小红  chinese    92   2023

    索引的变形#

    stack 与 unstack#

    stack()：列转为行索引（列 → 行） unstack()：行索引转为列（行 → 列）

    案例

    df = pd.DataFrame(
    [[85, 90], [78, 92]],
    index=['小明', '小红'],
    columns=['数学', '语文']
)

print("原表：")
print(df)




# stack：列转为内层索引
stacked = df.stack()
print("\nstack()：")
print(stacked)

# unstack：内层索引转为列
unstacked = stacked.unstack()
print("\nunstack()：")
print(unstacked)


输出：

原表：
     数学  语文
小明  85  90
小红  78  92

stack()：
小明  数学    85
     语文    90
小红  数学    78
     语文    92
dtype: int64

unstack()：
     数学  语文
小明  85  90
小红  78  92

    聚合与变形的关系#

    • pivot_table 本质是 先聚合，再变形
    • groupby + unstack 可实现类似 pivot_table 的效果

    案例：等价操作

    df = pd.DataFrame({
    'A': ['a', 'a', 'b', 'b'],
    'B': ['x', 'y', 'x', 'y'],
    'C': [1, 2, 3, 4]
})

# 方法1：pivot_table
pt = df.pivot_table(index='A', columns='B', values='C', aggfunc='sum')

# 方法2：groupby + unstack
gb = df.groupby(['A', 'B'])['C'].sum().unstack()

print("pivot_table:")
print(pt)
print("\ngroupby + unstack:")
print(gb)


输出一致：

B  x  y
A
a  1  2
b  3  4

    其他变形函数#

    crosstab交叉表（频次统计）#

    功能：计算两个或多个变量的频数表。

    案例：性别与科目交叉表

    df = pd.DataFrame({
    '性别': ['男', '女', '男', '女'],
    '科目': ['数学', '数学', '语文', '语文']
})

ct = pd.crosstab(df['性别'], df['科目'])
print("crosstab 交叉表：")
print(ct)

    输出：

    科目  数学  语文
性别
女     1    1
男     1    1

    explode将列表元素展开为多行#

    功能：将一列中的列表/元组展开，每元素占一行。

    案例：学生兴趣班

    df = pd.DataFrame({
    '学生': ['小明', '小红'],
    '兴趣': [['篮球', '音乐'], ['绘画']]
})

df_exploded = df.explode('兴趣')
print("explode 展开：")
print(df_exploded)


输出：
   学生  兴趣
0  小明  篮球
0  小明  音乐
1  小红  绘画

    get_dummies独热编码（One-Hot Encoding）#

    功能：将分类变量转为多列 0/1。

    案例：科目独热编码

    df = pd.DataFrame({'科目': ['数学', '语文', '数学']})
dummies = pd.get_dummies(df['科目'], prefix='科目')
print("get_dummies 独热编码：")
print(dummies)


输出：


   科目_数学  科目_语文
0      1      0
1      0      1
2      1      0

    总结：何时使用哪种变形？#

    场景 推荐函数
    长表 → 宽表（无重复） pivot
    长表 → 宽表（有重复，需聚合） pivot_table
    宽表 → 长表 melt / wide_to_long
    列 ↔ 行索引转换 stack / unstack
    频次统计表 crosstab
    列表展开为行 explode
    分类变量编码 get_dummies
