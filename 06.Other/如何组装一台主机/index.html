    志强cpu装机事项

    • 1.先定 CPU
      • 1.1 E3 平台
      • 1.2 E5 平台
      • 1.3 双路cpu
      • 1.4 E5 CPU推荐表
    • 2.内存
    • 3.选择主板
    • 4.选硬盘

    👉 E5玩的不是CPU，是整个平台：主板 + 内存 + 电源 + 散热，千万别省电源。

    1.先定 CPU#

    1.1 E3 平台#

    • E3 v1 / v2：LGA1155 → H61 / B75 / Z77（需 BIOS 支持）
    • E3 v3 / v4：LGA1150 → 部分 B85 / Z87 / Z97 支持（看主板）
    • E3 v5 / v6：仅支持 C232 / C236 工作站主板

    Intel 为区分平台、保护 i7，不再放行消费级主板

    1.2 E5 平台#

    • E5 v1 / v2：LGA2011 → X79
    • E5 v3 / v4：LGA2011-3 → X99

    1.3 双路cpu#

    是否需要双路cpu：不建议

    • 双路的本质，是当年顶级CPU太贵，用两颗中端CPU拼性能的性价比方案，并不是性能翻倍方案。

    • 实际性能提升并非1+1=2，通常只有约1.3～1.5倍；适合建模、渲染等多核负载，不适合打游戏。

    • 同时双路对内存要求较高，建议使用DDR4，否则容易出现性能下降甚至掉帧。

    • 例如双 Intel Xeon E5-2680 v4 ≈ 单 Intel Xeon E5-2696 v4，本质就是当年的高性价比替代方案。

    1.4 E5 CPU推荐表#

    型号 核心/线程 内存支持 主频 / 睿频 模拟器多开 功耗（24h满载） 适合人群
    Intel Xeon E5-2666 v3 10核20线程 DDR3 / DDR4 2.9GHz / 睿频≈3.5GHz 6～10开 ≈3～4度电 单核强，适合少开/网游
    Intel Xeon E5-2673 v3 12核24线程 DDR3 / DDR4 2.4GHz / 睿频3.2GHz 8～10开 ≈3～4度电 入门多开 + 办公
    Intel Xeon E5-2686 v4 18核36线程 DDR3 / DDR4 2.3GHz / 全核≈2.7GHz 15～20开 ≈5度电 性价比首选
    Intel Xeon E5-2696 v3 18核36线程 DDR3 / DDR4 2.3GHz / 睿频3.6GHz 15～20开 ≈6～7度电 主频更高一点
    Intel Xeon E5-2696 v4 22核44线程 DDR3 / DDR4 2.2GHz / 睿频3.6GHz 20～25开 ≈6度电 🔥 单路最强

    👉 2666 v3：核心少，但主频高 → 适合游戏 / 轻度多开 👉 2673 v3：入门多开 → 便宜够用 👉 2686 v4：性价比王 → 工作室主力 👉 2696 v3：频率更高 → 比2686稍强但更费电 👉 2696 v4：单路天花板 → 直接一步到位

    👉v3 vs v4

    v3：22nm v4：14nm v4优势：更省电/温度更低/更稳定 结论：能上v4就别选v3

    2.内存#

    👉 DDR3 vs DDR4

    • 实际多开：差距很小
    • 👉 当前行情：DDR3性价比更高

    👉 建议先查 DDR3 / DDR4 价格 再决定平台

    组装一台电脑 :(CPU + 主板 + 内存) 三者必须完全匹配, 必须都支持ddr3或者ddr4；

    3.选择主板#

    • 插槽：LGA2011 / LGA2011-3 必须匹配 CPU
    • 芯片组：优先 C612 / C610
      • B85 / H81 / HM55 多为魔改芯片
    • 内存类型：DDR3 / DDR4 必须与主板版本一致
    • 供电相数：4+2 < 6+2 < 8+2，越多越稳
    • 必须四通道
      • 4×16G ＞ 2×32G
      • 双通道只能发挥 60%性能
    • DDR3注意
    • 不要用32G单条（发热大、速度慢）
    • 用16G×4最稳
    • 关于“鸡血”（超频）不建议
      • 提升很小
      • 功耗暴涨
      • 主板压力大

    原厂服务器主板 vs 国产 X79 / X99

    • 原厂服务器主板问题： BIOS 锁死，仅认特定 CPU / 内存 强制 ECC REG 无视频 / 音频输出 机箱、电源不通用
    • 国产 X79 / X99 优势： 支持普通 DDR3 / DDR4 BIOS 开放，可鸡血 USB / M.2 / HDMI / 音频直出 更适合个人多开 / 渲染 / NAS
    • 国产主板本质 = 自主 PCB + 拆机芯片 + 魔改 BIOS

    4.选硬盘#

    • 看是否支持 M.2 再决定 SATA / NVMe
    • 硬盘属于外设，永远最后买，随时可升级
